# Memoria-privacy-policy
隱私權政策 (Privacy Policy)
最後更新日期：2026 年 6 月 23 日

本應用程式Memoria（以下簡稱「本服務」）非常重視您的隱私權。本政策說明我們如何處理您的個人資料。

1. 資料收集與使用
本應用程式使用 Google AdMob SDK 顯示廣告。為了提供廣告服務與分析應用程式效能，Google AdMob 可能會收集您的廣告識別碼 (Advertising ID) 及相關診斷數據。這些數據僅用於廣告投放與應用程式效能優化，不會與您的個人身份連結。

2. 資料儲存
本應用程式的所有功能資料（如使用者輸入的內容）皆儲存於使用者裝置的本機儲存空間，本應用程式並未將任何使用者個人資料上傳至雲端伺服器。

3. 資料刪除要求
由於本應用程式之資料皆儲存於使用者裝置本機，若使用者欲刪除所有相關資料，僅需解除安裝本應用程式，或前往裝置系統設定中清除本應用程式的數據與快取即可，所有本地資料將被永久移除。

4. 聯絡方式
若您對本隱私權政策有任何疑問，歡迎隨時透過電子郵件與我們聯繫。
